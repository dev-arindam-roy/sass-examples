How to use sass ? just follow below steps..
  • go to https://sass-lang.com/
  • install npm first
  • then install sass by using this npm command - "npm install -g sass"
  • then check the sass version
  • Open your CMD and type sass --version
  • you will see the sass version
  • that means - your system has been successfully install the sass
  • now you ready to work with sass

How to build sass

  • now you create a folder for sass project
  • within that folder, you create "index.html" file & "css folder" & "sass folder"
  • in the css folder you have to create a css file - like style.css
  • in the sass folder you have to create a scss file - like style.scss
  • the link the css file with your index.html
  • now you write your all css code into style.scss file
  • for conver scss to css - run below command -
  • sass sass/style.scss:css/style.css
  • for continue update - run below command
  • sass --watch sass/style.scss:css/style.css
  • then auto create all scss code to css
